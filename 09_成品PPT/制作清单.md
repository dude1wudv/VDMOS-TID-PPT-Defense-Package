# PPT 制作清单

- **最终页数**：74 页
- **页面比例**：16:9（10 × 5.625 inch）
- **风格**：蓝橙工业科技风，沿用原 PPT 配色体系
- **字体**：微软雅黑（标题 24pt 加粗，正文 14–16pt）

## 原页保留情况

- 母版与配色基础来自原讲稿版 PPT
- 全部 74 页按完整制作方案重新排版制作
- 关键实验图、仿真图使用交付包内 PNG 素材

## 页面结构概览

- **第一部分 研究背景、文献与实验方法**：1–10 页
- **第二部分 转移特性实验结果**：11–26 页
- **第三部分 输出特性与复扫现象**：27–40 页
- **第四部分 T 器件 TCAD 建模与验证**：41–53 页
- **第五部分 S 器件 TCAD 建模与验证**：54–59 页
- **第六部分 AI 流程、创新点与结论**：60–74 页

## 科学口径遵守情况

- [x] S = SGT VDMOS，T = Trench VDMOS，命名全篇一致
- [x] 每组 3 只器件，七剂量纵向跟踪，不称为独立样本
- [x] V@1µA 标注为高压代理指标，不等同标准 BV
- [x] Not/Nit 标注为等效参数化模型，非直接测量陷阱密度
- [x] S 组复扫称为扫描历史相关响应，不断言唯一机制
- [x] T 七剂量为共同标定，不称盲测预测
- [x] SS、D30/D40、高压求解边界等限制均保留
- [x] AI 定位为受监督的工程流程，不称自主发现物理规律
